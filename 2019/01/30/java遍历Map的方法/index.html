java遍历Map的方法

发表于 2019-01-30 更新于 2021-09-07 分类于 后端开发语言 ， Java ， 知识点

前言

遍历Map在Java中有好几种方式，但各自方式都有优缺点，不同的方式应用在不同的应用场景中

遍历方法

keySet values

在使用的过程中只是用到了Map的key或者value，这种是最简单的方式

keySet get(key)(不推荐使用)

如果同时需要key和value，可以先获取key，然后通过key获取到value，但这种方法不推荐使用

entrySet

通过map的entrySet进行遍历，是可以同时拿到key和value的，性能上更好一点，一般是用这种方式获取

Iterator

Iterator的优势是可以在遍历过程中删除元素

Lambda

Lambda是从java1.8开始支持的，语法更简洁，可以拿到key和value，但性能稍微低点

总结

  • 单纯获取key或者value，使用keySet
  • 需要同时获取key和value，使用entrySet
  • 遍历过程中如果需要删除操作，使用Iterator
  • 遍历过程中需要添加元素，可以新建一个临时map，遍历完成后，把临时map塞到最终map当中
