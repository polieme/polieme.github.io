Mock测试定义

Mock是在测试过程中，对于一些不容易构造/获取的对象，创建一个mock对象来模拟对象的行为；用人话或者个人理解来说，就是在前后端分离情况下，或者目标接口暂时没法测试的情况下，可以通过模拟返回数据的形式，进行初步的联调测试

RAP2

RAP2简介

RAP2 是在 RAP1 基础上重做的新项目，它能给你提供方便的接口文档管理、Mock、导出等功能，包含两个组件(对应两个 Github Repository)
