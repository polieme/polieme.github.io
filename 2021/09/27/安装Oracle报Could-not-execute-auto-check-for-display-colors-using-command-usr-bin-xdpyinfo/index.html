  2. 根据返回的结果，安装对应的包

  3. 只有xhost还不行，运行xhost+还是会报错，运行下面的命令安装vnc

  4. 启动并配置vnc

  5. 切换到oracle用户下

  6. 如果执行./runInstaller还是报错，则安装xdpyinfo

  7. 安装xdpyinfo

  8. 安装完成后执行./runInstaller就可以在Xshell下显示出安装页面了
